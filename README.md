# Italian Verb Practice (TypeScript)

A simple CLI to practice Italian verbs. Now migrated to TypeScript.

## Setup

```sh
npm install
```

## Build and run

```sh
npm run build
npm start
```

## Dev (run without build)

```sh
npm run dev
```

Data: `verbs.json` at the repo root. Progress is saved to `progress.json` (git-ignored).
